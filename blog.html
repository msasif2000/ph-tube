Q-1: Discuss the scope of var, let, and const.

Answer:

var: var is a keyword which is used to declare a variable in javascript. It is a global scope variable. It can be re-declared and updated.

let: let is a keyword which is used to declare a variable in javascript. It is a block scope variable. It can be updated but not re-declared.

const: const is a keyword which is used to declare a variable in javascript. It is a block scope variable. It can neither be updated nor re-declared.

Q-2: Tell us the use cases of null and undefined.

Answer:

null: null is a keyword which is used to assign a null value to a variable. It is used to empty a variable.

undefined: undefined is a keyword which is used to declare a variable without assigning any value to it. It is used to declare a variable without assigning any value to it.

Q-3: What do you mean by REST API?

Answer:

REST API: REST stands for Representational State Transfer. It is an architectural style for designing networked applications. It is a lightweight alternative to mechanisms like RPC (Remote Procedure Calls) and Web Services (SOAP) etc.
